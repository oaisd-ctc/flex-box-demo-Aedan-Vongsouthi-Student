Flex Container Tutorial

In webpages, CSS is used to format items and objects to suit how the webdesigner wants the webpage to look and feel.

Flex Containers can be used to easily section and seperate parts of your webpage, and in this tutorial, we will teach you how to create and use said things along with showing examples.

Flex Item Tutorial

Different, but still connected to Flex Containers, Flex Items change how the boxes are presented on a webpage instead of changing how it is structured/formatted.

Structure

Unlike Flex Containers, Flex Item Properties are usually used inline (without an External Stylesheet); You still need a Flex Container to use Flex Items, though, as Flex Items are meant to be used in conjunction to Flex Containers.

The structure for the code in HTML varies on what you want to do, but follows a basic structure.
